# WeChat Furniture Catalog Demo

这是一个已脱敏的微信小程序原生项目示例，用于展示家具/沙发图册类小程序的页面结构、筛选交互、详情页、活动页和联系页。

## 特点

- 原生微信小程序语法，不依赖 uni-app、Taro 等跨端框架
- 本地 mock 数据，可直接在微信开发者工具中导入运行
- 已替换真实产品图片、产品数据、品牌信息、地址、电话、公众号文章链接等敏感内容
- 保留主包 + 分包结构示例，适合继续替换为真实业务资源

## 运行方式

1. 使用微信开发者工具打开本目录。
2. AppID 可使用测试号或在 `project.config.json` 中替换为自己的 AppID。
3. 如需使用真实图片和数据，请替换 `data/`、`assets/images/` 和 `pkg/p1/assets/` 下的示例内容。

## 目录说明

- `pages/`: 首页、产品图册、活动、联系我们等页面
- `components/`: 产品卡片、活动卡片、标题区块、咨询条等公共组件
- `data/`: 脱敏后的品牌、产品、活动和联系方式 mock 数据
- `assets/images/`: 主包公共图片与 tabBar 图标
- `pkg/p1/`: 产品详情分包示例

## 注意

本项目仅为开源演示版本，不包含真实品牌、产品、地址或联系方式。
